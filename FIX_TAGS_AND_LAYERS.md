# 修复标签和层级设置

## 🔍 错误信息

```
Tag: Hazard is not defined.
```

## ✅ 解决方案

### 方案1：使用自动工具（推荐）

已创建自动设置工具，可以一键添加所有需要的标签：

1. **打开Unity编辑器**
2. **运行标签设置工具**
   - 在Unity顶部菜单栏，点击 `Tools`
   - 选择 `设置项目标签和层级`
   - 工具会自动添加所有需要的标签

3. **检查结果**
   - 查看Console窗口，应该看到成功消息
   - 警告应该消失

### 方案2：手动添加标签

如果自动工具不工作，可以手动添加：

#### 操作步骤：

1. **打开标签和层级设置**
   - 在Unity顶部菜单栏，点击 `Edit`
   - 选择 `Project Settings...`
   - 在左侧列表中选择 `Tags and Layers`

2. **添加标签**
   - 在 `Tags` 部分，找到 `+` 按钮
   - 点击添加新标签
   - 依次添加以下标签：
     - `Ball` - 小球标签
     - `Goal` - 终点标签
     - `Checkpoint` - 检查点标签
     - `Hazard` - 危险区域标签

3. **保存设置**
   - 设置会自动保存
   - 关闭Project Settings窗口

## 🔧 详细操作步骤（手动方法）

### 步骤1：打开Project Settings

1. 在Unity顶部菜单栏
2. 点击 `Edit`
3. 选择 `Project Settings...`
4. 窗口会打开

### 步骤2：选择Tags and Layers

1. 在Project Settings窗口左侧
2. 找到并点击 `Tags and Layers`
3. 右侧会显示标签和层级设置

### 步骤3：添加标签

1. **找到Tags部分**
   - 在窗口右侧，找到 `Tags` 部分
   - 应该能看到一个列表，显示现有标签

2. **添加Ball标签**
   - 找到列表下方的输入框（显示"Element 0"或类似）
   - 或者找到 `+` 按钮
   - 输入：`Ball`
   - 按回车确认

3. **添加其他标签**
   - 重复上述步骤，添加：
     - `Goal`
     - `Checkpoint`
     - `Hazard`

4. **检查标签列表**
   - 确认所有4个标签都已添加
   - 标签顺序不重要

### 步骤4：验证

1. **检查Console窗口**
   - 警告应该消失
   - 如果有其他错误，查看错误信息

2. **测试场景**
   - 打开游戏场景
   - 点击Play按钮
   - 检查是否还有标签相关的警告

## ⚠️ 注意事项

### 标签命名规则
- 标签名称区分大小写
- 必须与代码中使用的完全一致
- 不能包含空格或特殊字符

### 标签数量限制
- Unity默认最多32个标签
- 通常足够使用

### 如果标签已存在
- 如果尝试添加已存在的标签，Unity会提示
- 可以忽略，继续添加其他标签
